Description
This library let you scan Class and can change javascript Object into "Impostor".


Fonctionnement
The PorteMod copies methods from Class and store them at a IDAddress.
Then, the PorteMod can modify an Object by changing all its methods with a hook function and pair a FunctionalAddress to the modified object.
Finally, you associate IDAddress to FunctionalAddress and the Impostor object will change behavior.

Usages
Update running code without shuting down the program.
Enable Aspect Programing by intercepting method.

How to install

Copy the library into your project and instantiate a PorteMod and you are ready to go. Look at example files if you are lost.
